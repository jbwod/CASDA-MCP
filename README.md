# CASDA MCP Server

`casda-mcp` provides structured, auditable Model Context Protocol tools for the CSIRO ASKAP
Science Data Archive (CASDA). Metadata search and inspection are read-only. Archive-side staging
and local downloads are separate operations and are disabled by default.

This project is under active initial development. See `casda-mcp --help` for the current command
surface; complete installation, security, operations, and tool documentation is added with the
production deployment checkpoint.
